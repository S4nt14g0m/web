Safety when on the Internet

Internet security is all those precautions taken to protect all
the elements that are part of the network, such as infrastructure
and information,which is usually the most affected by cybercriminals.

Dangers of the internet

Scams.
Purchases induced by abusive advertising.
Purchases by minors without parental authorization.
Thefts.
Criminal actions for violation of intellectual property.
Carrying out illegal businesses.
Exorbitant telephone costs.

Security on your devices

Mobile device security is the protection of data on portable devices and to network
to which they are connected. Here are some tips to protect your mobile devices:
Keep software up to date
Use strong passwords
Activate auto lock
Avoid using public networks
Download apps only from trusted sources
Review app permissions
Take backups regularly
Use two-factor authentication
Set screen lock
Install security software

How to protect yourself from cyber attack

Also install antivirus and antimalware programs. Use good passwords:
When using passwords,is better to use variants that are always different
and with numerous characters. If they are long and have lowercase letters,
uppercase letters, numbers and special characters, much better.
